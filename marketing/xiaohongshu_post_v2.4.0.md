## 正文

扫了一整卷 135 胶片，最烦的是什么？

不是扫描，是在 Lightroom 里一帧帧手动拉裁剪框。

一卷发过来 36 张，手都要断了。

所以我做了这个小工具 👇

**NegativeCutter-135**

一个 Lightroom Classic 插件，自动识别长条扫描图里的每一帧，直接创建带精确裁剪的虚拟副本。

✅ 支持批量处理
✅ 自动检测帧数
✅ 0.3% 边界清理，去掉扫描脏边
✅ 黑白负片识别效果最好

⬇️ 安装超简单

下载 → 双击 install.sh → 重启 Lightroom → 菜单里找到 "检测胶片帧"

不需要装 Python，不需要 pip，开箱即用。

🎞️ 目前只支持 135 胶片规格，120 和 110 在开发了。

💬 有 bug 或想要的功能可以留言，我会看。

🔗 下载地址见评论区 / GitHub
